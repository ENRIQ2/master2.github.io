LISTADO ETIQUETAS HTML

    ETIQUETAS INICIALES

  1. <!DOCTYPE HTML> Indica al navegador que el documento está basado en el estándar HTML5

  2. <html> </html> Representa la raíz de un documento HTML. Todos los demás elementos de la estructura HTML deben ser recogidos dentro de estas etiquetas.

  3. < head> </head> Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. El resto de etiquetas de metadatos, irán recogidas dentro de las etiquetas de apertura y cierre del head. Importante explicar que estos metadatos del documento, es información para el navegador y no contenido que será visible en la página web. A excepción de la etiqueta < title> que veremos a continuación.

  4. < title> </title> Etiqueta usada para definir el título de la página web.

  5. < link> Se usa para enlazar recursos externos al documento HTML. El ejemplo más común son las hojas de estilos CSS.

  6. <meta> Etiqueta usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo para definir el autor del sitio, o la descripción del mismo.

  7. < style> </style> Etiquetas usadas para introducir código CSS en línea, es decir, en el propio documento HTML.

    8. ETIQUETAS DE SECCIONES

  8. <body> </body> Al contrario que la etiqueta de metadatos <head>, todo lo que quieras mostrar en la página web debe ir recogido dentro de las etiquetas de apertura y cierre de <body>. Este contenido será el que se muestre en la web.

  9. <nav> </nav> Usadas para definir el contenido que será la sección de navegación de la web.

  10. <main> </main> Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.

  11. <section> < /section> Define una sección del documento

  12. <article> </article> Define contenido independiente de la web.

  13. <aside> </aside> Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia

  14. < h1>, < h2>, < h3>,< h4>, <h5>, <h6> Son etiquetas HTML muy importantes, ya que son usadas para jerarquizar el contenido de la web. Las etiquetas se usan para explicar brevemente el contenido que irá a continuación.

  15. < header> </header> Se usan para definir la cabecera la página web. Suele contener el logotipo, menú de navegación, etc.

  16. < footer> </footer> Usadas para definir el pie de página.

    17. ETIQUETAS PARA LA AGRUPACIÓN DE CONTENIDO

  17. <em> </em> Fragmento de texto enfatizado respecto a la frase que lo contiene.

  18. <i> </i> Fragmento de texto con voz o tono alternativo al resto.

  19. <b> </b> Fragmento de texto sin importancia destacable

  20. <u> </u> Fragmento de texto para nombres propios o escritura incorrecta intencionada (sic).

  21. <s> </s> Fragmento de texto inexacto o que ya no es relevante.

  22. <abbr> </abbr> title Abreviatura o acrónimo.

  23. <p> </p> Etiqueta usada para escribir párrafos de texto.

  24. <hr> </hr> Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.

  25. < pre> </pre> Usada para pegar texto manteniendo el pre formato propio del texto.

  26. <blockquote> </blockquote> Se usan para indicar que el contenido es texto citado.

  27. <ol> </ol> Etiquetas para crear una lista ordenada

  28. < ul> </ul> Etiquetas para crear una lista des-ordenada

  29. <li> </li> Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.

  30. <dl> </dl> Usada para crear una lista de definiciones.

  31. <dt> </dt> Representa un término definido por la siguiente etiqueta <dd>

  32. <dd> </dd> Se usa para definir los términos listados antes que él.

  33. <figure> </figure> Indica una figura ilustrada como parte del documento HTML5.

  34. <figcaption> </figcaption> Utilizada para definir la leyenda de una figura.

  35. <div> </div> Etiqueta común utilizada para crear un contenedor genérico.

    36. ETIQUETAS SEMÁNTICAS PARA TEXTO

  36. <a> </a> Etiqueta utilizada para crear hiperenlaces en el documento HTML

  37. <wbr> </wbr> Oportunidad de salto de línea

  38. <strong> </strong> Etiqueta para definir una palabra o conjunto de ellas como importantes. Tiene una fuerte importancia en el SEO de la página.

  39. <small> </small> Utilizada para dejar un comentario aparte, del tipo una nota de derechos de autoría, u otros textos que no son esenciales para la comprensión del documento.

  40. < cite> </cite> Para indicar el título de una obra

  41. <sub> </sub> Etiquetas utilizadas para representar un subíndice o superíndice.

  42. <sup> </sup> superíndice.

  43. <mark> </mark> Usada para resaltar texto

  44. < span> </span> Etiqueta HTML sin ningún significado específico. Se usa conjuntamente con los atributos «class» o «id» para atribuirle ciertas características.

  45. <br> Etiqueta utilizada para crear un salto de línea

    46. ETIQUETAS PARA INCRUSTAR CONTENIDO

  46. <track></track> Establece un archivo de subtítulos o lo añade como alternativa.

  47. <iframe> </iframe> Permite incrustar contenido externo en «vivo».

  48. <param> </param> Define parámetros de un elemento <object>

  49. <img> Etiqueta para «pintar» una imagen en la página web.

  50. <picture> <picture> Agrupa una serie de imágenes. Etiqueta contenedora.

  51. <iframe> </iframe> Es una etiqueta que sirve para anidar otro documento HTML dentro del documento principal.

  52. <embed> Usada para integrar una aplicación o contenido interactivo externo que no suele ser HTML.

  53. <object> </object> Utilizada llamar a un recurso externo de la web. Este recurso será tratado como una imagen, o un recurso externo para ser procesado por un plugin.

  54. <video> </video> Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.

  55. <audio> </audio> Usada para cargar en una web un archivo de audio o stream de audio.

  56. <source> </source> Permite a autores especificar recursos multimedia alternativos para las etiquetas de <video> o <audio></audio>

  57. <svg> </svg> Se usa para llamar a una imagen vectorizada.

    58. ETIQUETAS PARA LA CREACIÓN DE TABLAS

  58. <table> </table> Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla han de ir siempre recogidas entre estas dos etiquetas.

  59. <caption> </caption> Usada para indicar el título de la tabla.

  60. <colgroup> </colgroup> Etiqueta utilizada para agrupar dos o más columnas de una tabla.

  61. <tbody> </tbody> Usada para describir los datos concretos de una tabla.

  62. <thead> </thead> Indica el bloque de filas que describen las etiquetas de las columnas de la tabla.

  63. <tfoot> </tfoot> Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla.

  64. <tr> </tr> Se usa para indicar una fila de celdas de una tabla

  65. <td> </td> Usada para definir una celda de una tabla.

  66. <th> </th> Etiqueta que se usa para definir el encabezado de una celda

    67. ETIQUETAS PARA LA CREACIÓN DE FORMULARIOS

  67. <form> </form> Etiqueta de apertura y cierre de un formulario de una página web. El resto de etiquetas de formulario deben ir siempre recogidas entre estas etiquetas de apertura y cierre de formulario.

  68. <fieldset> </fieldset> Etiqueta que representa un conjunto o agrupación de elementos de un formulario. «Pinta» un recuadro alrededor de las etiquetas que estén contenidas dentro del <fieldset> </fieldset>

  69. <legend> </legend> Etiqueta ligada a <fieldset>. Indica el título del <fieldset></fieldset>

  70. <label> </label> Se usa para definir el nombre o título de un control del formulario.

  71. <input> Pinta un campo de introducción de datos para el usuario. Es de las principales etiquetas de un formulario.

  72. <button> </button> Etiqueta utilizada para representar un botón en el formulario.

  73. <select> </select> Input que permite una selección entre un conjunto de opciones.

  74. <option> </option> Etiqueta ligada a <select>. Permite añadir diferentes opciones al <select> </select>

  75. <textarea> </textarea> Añade un campo al usuario para que pueda introducir texto en unas líneas máximas de texto que el desarrollador puede definir.

    76. ETIQUETAS OBSOLETA

  76. <applet> </applet> Etiqueta para applets Java.

  77. <acronym> </acronym> Indica un acrónimo.

  78. <bgsound> </bgsound> Especifica un sonido de fondo.

  79. <dir> </dir> Indica una lista de archivos o carpetas.

  80. <frame> </frame> Define un marco específico.

  81. <frameset> </frameset> Define un conjunto de marcos.

  82. <noframes> </noframes> Indica una alternativa si el navegador no soporta marcos.

  83. <hgroup> </hgroup> Agrupa encabezados.

  84. <isindex> </isindex> Campo para búscar en el documento.

  85. <listing> </listing> Fragmentos de código fuente.

  86. <noembed> </noembed> Alternativa (fallback) para contenidos.

  87. <strike> </strike> Muestra un texto tachado.

  88. <basefont> </basefont> Define una tipografía por defecto.

  89. <big> </big> Aumenta el tamaño del texto.

  90. <blink> </blink> Muestra el texto de forma parpadeante.

  91. <center> </center> Centra el texto.

  92. <font> </font> Cambia la tipografía o sus características.

  93. < marquee> </marquee> Muestra el texto moviéndose de un lado a otro.

  94. < multicol> </multicol> Columnas múltiples.

  95. <nobr> </nobr> Evita que un texto haga un salto de línea.

  96. <spacer> </sparce> Inserta un espacio horizontal.

  97. <tt> </tt> Muestra el texto con una fuente monoespaciada.

  98. <menu> </menu> Crea menús de listados.

    99. ETIQUETAS HTML INTERACTIVAS

  99. <details> </details> Crea un elemento desplegable. Etiqueta que contiene todo el contenido del elemento.

  100. <summary> </summary> Título del desplegable que aparece siempre, se encuentre desplegado o no.

  101. <dialog> </dialog> Ventana de diálogo que puede contener y mostrar cierta información.

  102. <kbd> </kbd> Entrada de información del usuario (generalmente, combinación de teclado).
